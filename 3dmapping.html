3D模型投影Mapping

建筑外墙、实体模型、数字沙盘三维曲面精准投影

适用场景

Outdoor Building/Car、plane model projection/City model show

户外楼体灯光秀、汽车/飞机实体模型展示、城市数字沙盘、异形雕塑投影

核心功能介绍

1. 标准OBJ三维模型兼容

Rending the 3D model file in Wavefront obj format，which can be exported by most of the popular 3D modeling software，such as 3DMax or Blender。

支持3ds Max、Blender等主流三维软件导出Wavefront OBJ模型，自带纹理贴图坐标，直接导入软件使用。

2. 多通道同步播放

Support multiple outputs in single machine， or multiple outputs with multiple machines

单机多输出通道、多主机分布式同步双架构，适配大型户外楼体项目。

3. 实时渲染，无需预生成视频

Textured video is rendered in realtime，no need preprocessing。

视频内容实时映射渲染，不用提前分通道预渲染影片，替换素材简单高效。

4. 布局方案保存复用

Each video window is maped to a projector/display。All windows are saved to a window layout set。Can be recalled anytime，and only video file is changed when you want to play another video

不同播放内容对应独立开窗布局，布局文件一键保存加载，切换视频无需重新校正。

5. 投影机任意摆放

Projector/screen can be aligned randomly，horizontally or vertically。

投影机横投、竖投、倾斜架设均可适配，无安装角度限制。

6. 后期图层处理

Post-processing such as edge-blending or masking can stilled be applied。

内置图层羽化、边缘融合、通道光闸遮罩，适配复杂建筑边角、遮挡场景。
