uvMapp 自动几何校正融合

uvMapp is a Camera based technique， for mapping the stage(multiple projection screens) coordinates into the scene(what we see) coordinates。With this technique，auto geometry registration and picture blending can be helpful to our technical staffs，no more handful work is needed。

uvMapp基于相机成像技术，一次获得舞台平面与投影光路的坐标关系，自动进行画面对齐和图像融合，无需过多人工网格微调，大幅降低现场调试工作量。

3D Mapping 三维模型投影

Playing the 3D model file in Wavefront obj format，which can be exported by most of the popular 3D modeling software，such as 3DMax or Blender。The projection screen is some structural building or 3D printing models with the same size of 3D model files。

支持3dmax、Blender导出标准OBJ三维模型，实体白模可等比例加工/3D打印，实现精准三维曲面投影映射。
